Both User Experience and the User Interface are not just important and necessary for the user, but also important for the designer themself. “The UX perspective takes this shift seriously. Its focus on aspects beyond the functional, on the positive, the experiential and emotional is no coincidence” [2]. The UX and UI design allows the designer to express themself and tries to transfer these expressions and passions to the user through both these elements. The content itself attracts the user but it is the UX and UI of the website that makes the user want to explore more of the website. It is once again through the good use of semantic markup, HTML, CSS, and JavaScript that allows the browser to help communicate what the designer intended and experience to the user. The continuous iteration of my UX and UI designs will continue to be developed until an optimal level of designer perspective is passed onto the user.
